Main Concepts Classes Are Everywhere - About Services

  • Angular Services are also only ES6 classes
  • Below is a naive implementation of a service to manage logs
  • Once the service is done, it has to be declared to the framework
enum LogLevel { INFO, WARN, ERROR } class LoggerService { info(msg: String): void{ if (this.minLevel >= LogLevel.INFO){ console.log('[INFO]' + msg); } } warn(msg: String): void{ if (this.minLevel >= LogLevel.WARN){ console.log('[WARN]' + msg); } } error(msg: String): void{ console.log('[ERROR]' + msg); } }

it is a best practice to separate presentation of data from data access by encapsulating data access in a separate service and delegating to that service in the component
