Main Concepts Router - Configuration And Usage

Things to see in the previous example

  • The import of RouterModule from @angular/router
  • The routeDefs array which defines our routes, where each route is defined by a path, and associated either to a component, either to a redirection to an other route
  • The forRoot method which allows to configure our app
  • Finally we use our directives:
    • the routerLink to go to a view by giving it the path to use
    • the router-outlet to specify where our view will be loaded
  • You can see that our routing definitions are in a specific file, named app.routing.ts, this is not mandatory, but is a convention suggested by Angular
