Preamble Angular VS AngularJS, what remains? How to upgrade?

To port from AngularJS to Angular, nothing magic, 3 important things:

  1. Prepare the existing application! Which means follow best practices, make it use recent technologies such as typescript/ES6, and use the Angular router ported to AngularJS
  2. Use AngularJS directives, which will ease the process of converting AngularJS directives to Angular components
  3. Include Angular in your AngularJS application, to make them coexist together and convert things incrementally (more for developments, not sure using two frameworks at the same time in production is a good idea)
