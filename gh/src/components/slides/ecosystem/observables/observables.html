Ecosystem A word about Observables

Observables are kind of advanced Promises and are used in Angular at different points

  • The EventEmitter allows to emit custom event between components (let's see it later) and are based on Observables, (even if you don't have to deal with complex concepts to use it)
  • The Http service returns an Observable to let you deal with an http request
  • This way, we can say that Angular encourages to use observables, but don't force you
