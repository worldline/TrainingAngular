Main Concepts PW3 - Routes & components

Let's create a new route and a new component "Contacts" to our application!

  • Create a new component Contact ( create the src/app/contact folder to do so)
  • Routes are defined in the App component (so src/app/app.routing.ts). Add a new route to Contact:
    • Name of the route: 'Contact'
    • Path of the route: '/contact'
  • In the template of the app component, there is a link to Contact which "does nothing", do the necessary to make it navigate to your new Contact component
  • Complete the Contact component:
    • Our contact should look like this (we don't care about css):
    • Add the fields mail/phone/postalMail in the Contact class and use them in the template
    • Use a textarea element to let the user write a message
    • Add an updateTextContent method in your component class, and call this method each time the text is modified in the textarea (by using the input event). The method does nothing, just be sure that you can access the textarea message from this method...
    • Add a click button which listen for click events and call a method. Again, the method does nothing for now.
