Advanced Concepts e2e testing

  • End 2 End testing is done in Angular as in AngularJS, with protractor
  • And there is no real difference in its use
  • The first things to do are to install local dependencies in your project and then to configure it
$> npm install protractor --save # You can also install it globally if you prefer $> npm install -g protractor exports.config = { //The selenium server drives your browser and // need to be contacted by protractor seleniumAddress: 'http://localhost:4444/wd/hub', //Same than unit tests framework: 'jasmine', // Define the specs path / pattern specs: [ 'app/testing/e2e/*.spec.js' ], capabilities: { 'browserName': 'chrome', 'version': 'ANY' }, //The base url baseUrl: 'http://localhost:8080', // Special option for Angular, to test against all AngularJS applications // on the page. This means that Protractor will wait for every app to be // stable before each action, and search within all apps when finding // elements. useAllAngular2AppRoots: true };
