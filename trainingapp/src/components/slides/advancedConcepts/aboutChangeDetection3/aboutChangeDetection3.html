Advanced Concepts Change Detection - Intro - When and Where

Where is a little bit more difficult to answer

  • It depends on the strategy used to detect changes, which is parameterizable at the Component level @Component({ //... changeDetection: ChangeDetectionStrategy.OnPush })
  • The available strategies are:
    • CheckAlways: default value, each time it has to check, the whole component tree is checked
    • OnPush: Angular will limit the observations in the tree to components inputs which have changed
