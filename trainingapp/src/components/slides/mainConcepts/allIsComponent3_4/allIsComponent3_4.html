Main Concepts Classes Are Everywhere - Directives / Components - Templates Overview

  • The example below is a component with its template, which also contains a reference to another directive
  • To make it work, you need to declare the nthclicks directive in @NgModule
@Component({ selector: 'hello-world', templateUrl:'hello-world.html', inputs: ['name'] }) class Redisplay{ private defaultName: string= 'Bob'; public name: string='Bob'; //default value constructor(){} resetName(){ this.name= defaultName; } } @NgModule({ imports: [BrowserModule], declarations: [ AppComponent, NbClicksDirective ], bootstrap: [ AppComponent ] }) class AppModule { } <h1>Hello World Template Content</h1> <p> I'm used to say hello to somebody, so hello {{name}}! </p> <button (click)="resetName()">Click Me To reset the name</button> <div class="redBox" [nthclicks]="3" #nthClicked="nthClicked"> Try to triple click me!!! <span *ngIf="nthClicked.done"> Hey, you did it </span> </div>
