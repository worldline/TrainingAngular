Main Concepts Pipes - Exisiting Ones - DatePipe

There is a number of existing pipes provided by Angular

  • To format a date
  • Type name: DatePipe
  • Expression name: date
  • Parameters: format
import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; @Component({ selector: 'app', template: `<p> Now: {{ now() | date:'dd/MM/yyyy' }} </p>` }) export class App { now(): Date { return new Date(); } } export const ModuleDefs= { bootstrap: [App], imports: [BrowserModule], declarations: [App] }
