Preamble Practical Work - Add your own Router module

  • Use the Angular router to enable navigation. The Angular router is an external, optional Angular NgModule called RouterModule.
  • The router is a combination of multiple provided services (RouterModule), multiple directives (RouterOutlet, RouterLink, RouterLinkActive), and a configuration (Routes).

  • Create a specific file named app.routing.ts that will define your app Routes.
  • Import that module into you application.
  • add a router-outlet element at the end of your app.component template.
  • add two routerLink elements at the top of your app.component template in order to redirect the user to a specific component.
