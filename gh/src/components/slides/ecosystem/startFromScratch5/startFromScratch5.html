Ecosystem Start From Scratch - Play quickly

A good way to test something with Angular is to use the typescript.js transpiler at runtime, because you don't need to install anything to use it. But again, if this is perfect for POCs, it's not recommended at all for production purposes...

Here is a minimal application sample :

  1. The index page is loaded
  2. Some polyfills are loaded
  3. The SystemJs module loader is configured
  4. At this point, everything needed to run an application is there
  5. The application starting point is loaded: 
    System.import('app/main.ts')
