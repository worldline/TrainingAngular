Main Concepts About Templates - Property Binding - the basics

In previous examples, we used the inputs attribute in the @Component annotation to declare a parameter for a component


With Typescript you can also use the @Input annotation as well


Let's try to set a name property to our custom SubCmp component:

import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'app', template: '<sub-cmp name="john"></sub-cmp>' }) class App {} export const ModuleDefs= { bootstrap: [App], declarations:[ App, SubCmp], imports: [BrowserModule] } import {Component, Input} from '@angular/core'; @Component({ selector: 'sub-cmp', template: 'Hello {{name}}' }) export class SubCmp { @Input() name: string; }
