Preamble Angular VS Polymer

  • Polymer is not really a framework, unlike Angular
  • Polymer gives you a set of components and allows anybody to import them in its own application
  • So Polymer is not really done to do SPAs, with stuff like routing and lifecycles features...
  • It's more a library to make Web Components and can be used for example with Meteor Js in order to add framework features
  • Finally, Polymer components can be used inside Angular applications as well...
