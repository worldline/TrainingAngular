Main Concepts About Templates - What about 2 way data binding?

  • In AngularJS, 2 way data binding is an usefull feature which allows to bind data from an input.
  • What does it do: update the model from the input, and then update the view from the model, everything at the same time
  • The following seems to do what we need: Name: <input type="text" [value]="name" (input)="name= $event.target.name" >
  • Seems to be fine, but a shortcut syntax has been defined: [(ngModel)]="field"
  • Of if you don't like the [(ngModel)] syntax, use bindon-ngModel instead
import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; import {FormsModule} from '@angular/forms'; @Component({ selector: 'app', template: ` <h1> Name: {{name}}</h1> <input type="text" [(ngModel)]="name" > ` }) class App { name= 'toto'; } export const ModuleDefs= { bootstrap: [App], declarations:[ App], imports: [BrowserModule, FormsModule] }

You have to import FormsModule in your one to use ngModel
