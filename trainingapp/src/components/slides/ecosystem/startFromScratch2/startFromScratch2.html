Ecosystem Start From Scratch - which language ?

  • As we told before, you can use typescript but it's not mandatory
  • If you prefer, you can use js directly, but which version will you use: ES6 or ES5?
    • If you don't use typescript, we recommend using ES6. Writing Angular in ES5 is possible, but it will be a pain for developpers to learn libraries whose the aim is to get functionalities which are native with ES6 such as classes. And more generally, ES6 is also there to make things more cumfortable (classes, array functions, templating...). The only use case for using ES5 is people which don't want to learn ES6, but this is not usual!

Remember that using typescript is generally interesting for most projects
