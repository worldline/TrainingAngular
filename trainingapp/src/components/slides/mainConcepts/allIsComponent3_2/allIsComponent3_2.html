Main Concepts Classes Are Everywhere - Directives

  • So a directives is an ES6 classes which contains the @Directive annotation
  • This example counts the number of clicks on an element import {Directive} from '@angular/core'; /** * A directive to listen the number of clicks on an element * * ### Example * <h1>Test Directive</h1> * <button [nthclicks]="2" #myCounter="nthClicked" > Click Me twice </button> * <p *ngIf="myCounter.done">Twice clicked ok</p> */ @Directive({ selector: '[nthclicks]', inputs: ['nthclicks'], host: { '(click)': 'clickedOnce()' }, exportAs:'nthClicked' }) export class NbClicksDirective { nthclicks= 1; private nbclicks= 0; done= false; clickedOnce= () => { this.nbclicks++; this.done= (this.nbclicks >= this.nthclicks); } }
