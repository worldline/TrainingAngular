Ecosystem Typescript - Decorators

Finally, typescript also provides a way to create Decorators. This feature is very closed to Java annotations. They allow to add information or behavior arround a method, an attribute, a class or a function parameter.

Decorators are often used in Angular apps to declare components, pipes and some other specific things.

As an Angular developer, you generally won't have to declare your own decorators, but you can if you need. Below is a basic decorator example. You want to manage a cache for some kind of asynchronous method, let's say for a method which retrieves user data from the server

function f() { console.log("f(): evaluated"); return function (target, propertyKey: string, descriptor: PropertyDescriptor) { console.log("f(): called"); } } function g() { console.log("g(): evaluated"); return function (target, propertyKey: string, descriptor: PropertyDescriptor) { console.log("g(): called"); } } class C { @f() @g() method() {} } //f(): evaluated //g(): evaluated //g(): called //f(): called class Person{ constructor(public firstName, public lastName) {} } let cachedPersons: { [index: number]: Person } = {}; let CachePerson= function() { //the generics TypedPropertyDescriptor allows us to tell that // this decorator can only be set on a function with the function //specified as generic return (target:any, propertyKey:string, descriptor:TypedPropertyDescriptor<(id:number) => Promise<Person>> ) => { //Let's still use initial method so keep a reference on it let initialMethod = descriptor.value; //But let's redefine our own method descriptor.value= function(id:number):Promise<Person> { return new Promise<Person>( (resolve, reject)=> { if (cachedPersons[id] !== undefined){ resolve(cachedPersons[id]); } else{ initialMethod(id).then( (person: Person) => { resolve(person); } ); } } ); } }; }; class PersonHelper{ //We set our annotation there @CachePerson() getPerson(id: number): Promise<Person> { return new Promise<Person>( (resolve, reject) => { setTimeout(()=>{ resolve(new Person('firstName' + id, 'lastName' + id )); }, 5000); }); } } let personHelper= new PersonHelper(); //Will wait 5 secs personHelper.getPerson(1).then( (person: Person) => {console.log('ok'); } ); //Will be immediately resolved personHelper.getPerson(1).then( (person: Person) => {console.log('ok'); } );
