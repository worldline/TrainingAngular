Main Concepts Classes Are Everywhere - Components

  • A component is also an ES6 class, but annotated which @Component
  • The @Component annotation allows to add some metadata, such as the template/templateUrl ones
  • The template can use the public methods attributes declared in the class
  • The easiest way to display a component property is to bind the property name through interpolation. With interpolation, you put the property name or expressions in the view template, enclosed in double curly braces: {{name}}.
@Component({ selector: 'hello-world', template:'<span>Hello {{name}} </span>' }) class Redisplay{ public name: string='world'; constructor(){} }
