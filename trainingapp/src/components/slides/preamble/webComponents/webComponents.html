Preamble About Web Components

  • Angular wants to be compliant with "future" promising web technologies, such as Web Components
  • Web Components consist in several concepts and standards:
    • Templates
    • Shadow Dom
    • Custom Elements
    • Packaging
  • Web Components brings a web standards-based way to create reusable components using nothing more than vanilla JS/HTML/CSS.
    The result is less code, modular code, and more reuse in our apps.
