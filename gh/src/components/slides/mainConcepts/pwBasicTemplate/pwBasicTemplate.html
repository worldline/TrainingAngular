Main Concepts PW1 - Templates

The aim of this PW1 is to let you manipulate simple templates:

  • To initialize your app, if not already done, run:
    • npm install
    • If you don't have a global gulp executable :
      npm install -g gulp
  • To launch the Back end server, open a console in subfolder /express/ and Run npm run serve.
    The app will serve the REST Api here.
  • To launch the client Side server, open a console in subfolder /ng2ecom/ and Run npm run start. The app will automatically reload if you change any of the source files and will work as a proxy for rest API.
  • Access your application here
  • Have a look at the app folder structure to have an idea of where are the things you'll have to implement. Ask if you have any question!
  • You will have to use the {{expression}} in order to bind a value from the component into the template .
  • Browse to bookDetails.html
  • Complete the TODO_BASICS comments in the template (displays informations about the book)
  • Go to this url to check your devs: http://localhost:4200/#/book/1

If book is undefined or null, {{book?.xxx}} will do the job
